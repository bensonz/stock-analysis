# 📈 A股每日分析报告

Daily Chinese A-share stock analysis powered by AI.

## 关于本仓库

本仓库包含基于预设策略筛选的A股每日分析报告。分析内容包括：

- **基本面分析** — PE、PB、ROE、营收增长、利润率
- **技术面分析** — 价格走势、成交量、均线系统、支撑/阻力位
- **板块与政策** — 行业趋势、政策利好/利空
- **资金流向** — 机构动向、北向资金

## 评级说明

| 评级 | 含义 |
|------|------|
| ⭐⭐⭐ 强烈推荐 | 多重利好共振，建议积极关注 |
| ⭐⭐ 推荐 | 基本面/技术面良好，可考虑布局 |
| ⭐ 观望 | 存在不确定性，建议等待更好时机 |
| ⚠️ 回避 | 风险较高，不建议介入 |
